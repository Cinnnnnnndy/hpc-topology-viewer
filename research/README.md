# research/ — 昇腾超节点可视化 · 研究归档

本目录归档驱动本可视化项目的**研究与设计输入**资料（原始调研、结构化数据规格、概念/架构/3D 可视化稿、设计简报）。源应用（`src/`）在此研究基础上构建。

> 注：本目录文件含产品/品牌明文，与 `src/` 的反抓取（base64 编码）约定不同——属经用户确认后的研究归档件。

## 文件清单

| 文件 | 类型 | 内容 |
|---|---|---|
| `昇腾超节点-各级互联设计简报.md` | 设计简报 | 完整设计输入：目标、领域速成、视觉编码系统、视图清单、交互模型、来源与可信度（A/B/C）。 |
| `昇腾超节点-可视化输入规格.json` | 结构化数据 | 下游可视化的唯一数据源：抽象层级 / 实体 / 互联各级 / 并行映射 / 拓扑规模 / 视图预设 / 配色 token / 来源。 |
| `昇腾超节点-概念可视化报告.html` | 概念图（静态 SVG） | 抽象层级栈、互联带宽/时延阶梯、节点/Die 内部、并行→UB 映射、芯片路标、拓扑规模。 |
| `昇腾超节点-全栈整合架构图.html` | 架构图 | 全栈整合架构总览。 |
| `昇腾超节点-3D可视化报告.html` | 交互 3D（Three.js 单文件） | 机柜→节点→Die→UB 互联→进程/线程映射五视图，可旋转/缩放/点击。 |

## 数据可信度

- **A** 官方一手（官网 / 社区 / HC2025 keynote / 官方文档）
- **B** 媒体·社区交叉验证一致
- **C** 厂商公开解读 / 媒体口径（正式引用需标注）

数据缺口（如 16/32/64p 逐档交换层数）在资料中以显式占位标注，未作编造。
