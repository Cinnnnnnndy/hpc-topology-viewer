层NVIDIA / NCCL昇腾超节点说明

算法层不随硬件变

集合原语（Broadcast / Scatter / Gather / AllGather / Reduce / ReduceScatter / AllReduce / AllToAll）和算法（ring / tree / 多 channel / 开环拼接）都是逻辑层——NCCL 怎么在环上搬数据，昇腾 HCCL 也一样搬。硬件只决定这条环/树落到哪些物理链路上（UB 域内还是跨 rail），以及换轨发不发生。所以工作台里的「原语数据流 / Ring·Tree 展开」两边通用，「物理路径投影」才是随硬件变的部分。
